Nimbus is a geolocation based social media app that allows users to tag where they've been and see what their friends are up to. The iOS mobile client is built with React Native, Redux, and Exponent.js. The backend is powered by Node.js, Express, and Neo4j (a graph database).

Colorz.io is an intuitive color picking application built with React, Node, and MongoDB. Users can browse color palettes and easily copy hex code values to the clipboard. Users can also create their own color palettes and get suggestions about which colors go well together. Please allow up to 30 seconds for the site to load. It is throttled by Heroku's "free" plan, and loads slowly if no one has visited the site for a while.

Crypto Watchdog is an API monitoring service for cryptocurrencies. The node.js service runs on a VPS and monitors the top cryptocurrency exchanges to detect new listings. It will make intelligent buying decisions when given the opportunity. Users are alerted when the bot takes action by interacting with an attached Telegram bot. The code for this project is closed source and only available by request.
