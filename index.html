Nimbus is a geolocation based social media app that allows users to tag where they've been and see what their friends are up to. The iOS mobile client is built with React Native, Redux, and Exponent.js. The backend is powered by Node.js, Express, and Neo4j (a graph database).

Colorz.io is an intuitive color picking application built with React, Node, and MongoDB. Users can browse color palettes and easily copy hex code values to the clipboard. Users can also create their own color palettes and get suggestions about which colors go well together.

SDSUEvents is an event discovery application that I made for students at my university. It is built with Ruby on Rails, Bootstrap, and PostgreSQL. Please allow up to 30 seconds for the site to load. It is throttled by Heroku's "free" plan, and loads slowly if no one has visited the site for a while.
